Sintaxe do java script:

Objeto

Neste exemplo, o objeto é a pessoa, altura é sua propriedade:

          pessoa.altura;

Isto abaixo é uma função. Funções são ações dos objetos.

          pessoa.andar();

Ponto e virgula “;” é usado no final da linha para indicar o fim do comando.

Em javascript existem:

  • objetos internos (string, date, number)
  • objetos do browser (window.document)
  • objetos criados pelo programador.

O popup que você recebeu na sua tela ao acessar esta página é um objeto do seu navegador chamado window. foi usado a função alert para que o popup seja apresentado:
Este foi o comando utilizado:


        window.alert(window.document.title);
