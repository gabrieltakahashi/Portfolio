Manipuladores de eventos

Através das ações dos objetos interagimos com a página. É possivel usar eventos para engatilhar uma ação.
Por Exemplo:

 <script>
        function ligar(){
            document.getElementById("lampada").src = ("img/OnLamp.png");
        }
 </script>

Temos aqui uma funcao que altera um atributo src de um elemento html chamado “lampada”.


<img id="lampada" src="img/OffLamp.png" onclick="ligar()" />

aqui temos dentro do elemento chamado lampada um evento chamado onclick. Este evento executa a funcao ligar() quando o usuário clicar no elemento.

Veja na prática!

Legal, né? você deve ter percebido que apos a lampada ligar, tudo fica meio sem graça, pois não é possivel faze-la desligar de volta clicando nela... a única maneira de fazer isso seria recarregando a página. logo mudaremos isso. :)
