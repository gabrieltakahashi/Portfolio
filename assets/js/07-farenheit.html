Este é um exemplo simples de calculo em javasript. aperte f5 caso queira testa-lo novamente.

Nesses calculos, é sempre bom lembrar que os operadores * e / tem precedencia sobre - e +.

 <script>
        // equacao farenheit: Cº = 5/9 (F -32)
        var faren = prompt("escreva os graus em farenheit",50);
        var cels;
        cels = 5/9 * (faren -32);

        alert(faren + "\xB0 Farenheeit são " + cels + "\xB0 centigrados.");
    </script>
