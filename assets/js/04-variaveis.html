Variaveis

variáveis sao um lugar na memória do computador onde colocamos valores para serem manipulados. declaramos um nome para este espaço em JavaScript da seguinte maneira:

function ligar() {
    var primeiroNome = "ricardo"; // uma string
    var idade = 21;  // number
    var casado = true; // boolean
}

no exemplo acima vemos 3 tipos primitivos de variaveis.

(Observações pessoais: is up to you a forma que voce nomeia suas variaveis!
algumas regras que eu sigo:

  • Use nomes significativos: assim voce evita de no futuro ficar quebrrando a cabeca tentando descobrir o que a variavel ‘asd123’ esta fazendo no seu programa bugado!
  • Cuidado com caracteres especiais: pessoalmente, uso apenas letras e numeros para nomear variaveis, para evitar conflito com bibliotecas.
  • cuidado com as palavras reservadas: consulte a especificação da ECMA262 para verificar as palavras que não podem ser usadas para nomear suas variaveis.
  • camelCase: existem vários padrões para nomear as variaveis. blablabla
    por padrao, nomeio com letras maiusculas para nao confundir com as CONSTANTES, que geralmente sao nomeadas com letras em caixa alta.
